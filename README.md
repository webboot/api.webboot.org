## api.webboot.org

This the @webboot api hosted on https://now.sh as serverless functions.

all functionality exists in the api directory.

[html-docs](https://webboot.org/docs/api/)

#### install:
```bash
git clone https://github.com/webboot/api.webboot.org
```

#### development:
```bash
npm install -g now

cd api.webboot.org

now dev
```
